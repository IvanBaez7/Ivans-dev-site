Current Projects/ in progress

Intercessory Prayer Network

The Intercessory Prayer Network or IPN is a social media app that allows people to pray and interceed for others all over the world. It also allows the user to pray for nations. This project is still in the works but the vision is there, and the bulk of design is finished. The backend is being developed currently.

IPN utilizes leaflet's JavaScript mapping library and accesses tiles from ThunderForest via API's. If there are any questions, comments or suggestions, I'd love to hear from you. Collaborate?

Weather app

The weather app is a functional piece drawing on Openweather's API. I added the current time to it and decided to have the background change depending on the weather.
